
# Loading 下拉刷新
### 介绍

简单的数据信息的展示

###  基本使用
```html
   <mz-loading :init="loadData"> 列表 </mz-loading>
```
```javascript
const loadData = () => {
   // ...
};
```
## API

| 参数     | 说明                                                         | 类型    | 默认值    |
| -------- | ------------------------------------------------------------ | ------- | --------- |
| init     | 下拉刷新要执行的方法                                             | Function  | 无 |

## Slot
| slot名 | 说明           | 回调参数     |
|--------|----------------|--------------|
| -  | 内容插槽，默认插槽 | - |
